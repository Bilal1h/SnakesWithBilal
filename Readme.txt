Snakes With Bilal is a game made with PYTHON using pygame.
You Can Also Play It.
Move Controls =   Left Arrow , Up Arrow , Right Arrow , Down Arrow
Cheat Codes =     'A' for increasing snake length with out 
	            incrementing score.
	            'Q' for increasing code by 50.
	            'W' for increasing speed by 5.
	            'S' for decreasing speed by 1.
